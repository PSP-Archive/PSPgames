OpenBricks
==========

OpenBricks is a Open-source PSP Tetris-like game. It focuses on a heavily customizable gameplay and a good handling.

Features to come:
>Win/Lose event sounds
>Improved "wallkick"
>Options menu
>Scoreboard
>New game modes
>LevelUp text
